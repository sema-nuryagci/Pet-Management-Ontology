# Pet-Management-Ontology
# Evcil Hayvan Yönetimi Ontolojisi

Bu proje, evcil hayvanların sahiplik durumlarını ve veteriner randevularını takip etmek için geliştirilmiş bir OWL ontolojisidir.

## Proje Kapsamı
- Evcil hayvanlar (Pet), sahipler (Owner) ve veterinerler (Veterinarian) arasındaki ilişkileri tanımlar.
- Hayvanların yaş ve tür bilgilerini tutar.

## Kullanılan Teknolojiler
- **OWL 2 DL** (Ontology Language)
- **METHONTOLOGY** (Geliştirme Metodolojisi)
